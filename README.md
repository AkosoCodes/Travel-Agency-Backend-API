## 🏁 | Getting Started <a name="getting_started"></a>
These instructions will get you a copy of the project up and running on your local machine.

**<h3>Requirements:</h3>**
<ul>
<li>MongoDB</li>
<li>Backend API</li>
</ul>

**<h3>Guide:</h3>**

1. Download the source code
2. Run the Backend API
3. In the Angular CLI:
<br>>```npm install```
<br>>```ng serve```
4. Enjoy! :D
